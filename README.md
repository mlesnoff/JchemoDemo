# JchemoDemo

**JchemoDemo** is a Julia [project-environment](https://github.com/mlesnoff/JchemoDemo/blob/main/Project.toml) of tutorials 
on package [**Jchemo**](https://github.com/mlesnoff/Jchemo.jl). It illustrates the general syntax of basic functions of the package.

The contents of the project are regularly updated.  

**Contents**

1. **Examples on datasets with Jchemo** 

    - [**Html**](https://mlesnoff.github.io/JchemoDemo/docs/build/) 
    
    - [**Julia markdown scripts**](https://github.com/mlesnoff/JchemoDemo/tree/main/Ex/jmd) corresponding to the above Html 

Note: The datasets used in the examples are part of package [**JchemoData**](https://github.com/mlesnoff/JchemoData.jl) (that is 
contained in the [**JchemoDemo environment**](https://github.com/mlesnoff/JchemoDemo/blob/main/Project.toml)).

2. **Annexes**

    - [**Tips**](https://github.com/mlesnoff/JchemoDemo/blob/main/Misc/tips_config_julia.md) to start with Julia

    - [**Scripts**](https://github.com/mlesnoff/JchemoDemo/tree/main/Misc/src) about general Julia syntax (not using Jchemo)

    - [**Slides**](https://github.com/mlesnoff/JchemoDemo/tree/main/Misc/annexes) on chemometrics methods
