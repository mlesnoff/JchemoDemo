Grid-search

The best syntax to use function gridscore for ridge-based models (Rr, Krr, etc.) is to set the parameter lb outside of the generic argument pars defining the grid. In that case, the computation time is reduced [See the naïve and non-optimized syntax at the end of this script]. This is the same principle when definining the parameter nlv in LV-based models (eg. Plsr, Kplsr, etc.).
