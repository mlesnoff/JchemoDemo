Grid-search

The best syntax to use function gridscore for ridge-based functions (rr, krr, etc.) is to set parameter lb outside of the generic argument pars defining the grid. In that case, the computation time is reduced [See the naïve and non-optimized syntax at the end of this script]. This is the same principle when definining the parameter nlv in LV-based functions (eg. plskern, kplsr, lwplsr, etc.).

lb = 10.0.^(-15:.1:3)
model = rr()
res = gridscore(model, Xcal, ycal, Xval, yval; score = rmsep, lb)

Naïve syntax to use gridscore for ridged-based functions

model = rr()
res = gridscore(model, Xcal, ycal, Xval, yval; score = rmsep, pars)
