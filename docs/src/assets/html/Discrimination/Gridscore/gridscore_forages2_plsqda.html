Grid-search

The importance of each class is made equal, and a continuum between PLS-QDA and PLS-LDA is evaluated. 

julia> prior = [:unif]
1-element Vector{Symbol}:
 :unif

julia> alpha = [0, .25, .5, .75, 1]  # continuum parameter: from alpha = 0 (PLS-QDA) to alpha = 1 (PLS-LDA)
