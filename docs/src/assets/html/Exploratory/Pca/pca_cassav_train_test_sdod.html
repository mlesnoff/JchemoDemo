Pca - cassav - train vs test - sdod

When the dataset is split to a training and a test set, it can be relevant to check that the two sets share the same space. From a PCA, this can be done by plotting the score vs. orthogonal distances (SD and OD, respectively). This can be implemented for instance by using function occsdod that also returns a one-class classification (OCC) index as a compromise between SD and OD.
